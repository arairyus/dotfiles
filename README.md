## codespace

> 新しい codespace を作成すると、GitHub は選択したドットファイルのリポジトリを codespace 環境にクローンし、次のいずれかのファイルを探して環境を設定します。
> 
> install.sh  
> install  
> bootstrap.sh  
> bootstrap  
> script/bootstrap  
> setup.sh  
> setup  
> script/setup
